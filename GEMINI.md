# Project: Good Shepherd 2025 Theme

## Overview
Custom WordPress theme development environment using Vite and Tailwind CSS.

## Tech Stack
- **CMS:** WordPress
- **Bundler:** Vite (v7)
- **CSS:** Tailwind CSS (v4)
- **JS:** Vanilla ES Modules & jQuery

## Architecture & Workflow
- **Source:** `src/` (Edit files here)
    - `src/main.js`: Main entry point.
    - `src/style.css`: Main stylesheet (Tailwind imports).
- **Output:** `assets/` (Generated by build - DO NOT EDIT DIRECTLY)
    - `assets/js/`: Compiled JavaScript.
    - `assets/css/`: Compiled CSS.
- **Local Dev:** `https://goodshep2025.local` (Proxied via BrowserSync).

## Development Rules
1. **Vite Config:** Build output is customized to `assets/` (not `dist/`).
2. **Styling:** Use Tailwind utility classes whenever possible.
3. **PHP:** Ensure all PHP files are compatible with the latest WordPress version.

## Commands
- `npm run dev`: Start Vite dev server with BrowserSync proxy.
- `npm run build`: Compile assets for production.
